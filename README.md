# 🤖 AI Stress Support Bot (Zapier)

This project is an AI-powered chatbot built using Zapier Interfaces.

## 🌱 Features

* Emotional support and stress management
* Simple calming techniques (breathing, grounding)
* Multilingual (EN / CZ / UA)
* Works as a website chatbot

## ⚙️ Tech Stack

* Zapier Interfaces
* AI Assistant

## 🔗 Live Demo

https://sn-ai-content.zapier.app

## 💬 Embed Example

You can embed this chatbot into any website using:

```html
<script async type='module' src='https://interfaces.zapier.com/assets/web-components/zapier-interfaces/zapier-interfaces.esm.js'></script>
<zapier-interfaces-chatbot-embed is-popup='true' chatbot-id='cmosf9z6w0021kz9wlhlqudpu'></zapier-interfaces-chatbot-embed>
```

## 📌 Use Case

This bot helps users manage stress and provides simple emotional support in real time.
